# Mon Portfolio en Ligne 🚀

Bienvenue sur mon portfolio en ligne ! Ce site présente mes compétences, projets et expériences en développement web.

## 📋 Caractéristiques

✨ **Design moderne et responsive**
- Interface épurée et professionnelle
- Optimisé pour tous les appareils (mobile, tablette, desktop)
- Navigation fluide et intuitive

🎯 **Sections principales**
- **Accueil** : Présentation avec appel à l'action
- **À propos** : Informations personnelles et professionnelles
- **Projets** : Galerie de mes réalisations
- **Compétences** : Liste complète de mes savoir-faire
- **Contact** : Formulaire et moyens de me joindre

⚙️ **Technologies utilisées**
- HTML5
- CSS3 (avec variables et animations)
- JavaScript (Vanilla)
- Font Awesome (icônes)

## 📱 Responsive Design

Le site est entièrement responsive :
- Desktop : Layout complet avec navigation horizontale
- Tablette : Ajustements automatiques

## 🚀 Démarrage

1. **Cloner le repository** (ou télécharger les fichiers)
2. **Ouvrir `index.html`** dans un navigateur web
3. C'est tout ! Le site est prêt à l'emploi

## 📝 Personnalisation

### Modifier les informations personnelles
Ouvrez `index.html` et modifiez :
- Votre nom
- Votre email et téléphone
- Vos liens de réseaux sociaux
- Votre description

### Ajouter des projets
Dans la section `<!-- Projects Section -->` de `index.html`, dupliquez un `project-card` et modifiez :
- Le titre
- La description
- Les technologies utilisées
- Le lien vers le projet

### Personnaliser les couleurs
Dans `styles.css`, modifiez les variables CSS dans `:root` :
```css
:root {
  --primary-color: #2563eb;    /* Changez cette couleur */
  --secondary-color: #1e40af;
  --accent-color: #f59e0b;
  /* ... */
}
```

### Ajouter des sections
Créez des sections supplémentaires en copiant la structure :
```html
<section id="section-id" class="section-class">
  <div class="container">
    <!-- Contenu -->
  </div>
</section>
```
